## 4. 실행 방법

- 확인 url : <https://uuu25-pomodoro-timer.netlify.app/>
- 이전 세팅값 적용되어있음, 우측 상단 설정(톱니바퀴) 선택
- 원하는 시간(1~60 분) 입력 후 저장 > 메인으로 이동
- 시작 버튼 클릭 시 타이머 진행
- 종료 후 타이머 박스 shake 애니메이션 확인 > 박스 클릭시 멈춤 확인 가능

## 5. 만든 이유, 목표

- 간단한 기능 구현으로 클로드 코딩 활용법 실습
- 실습 내용 외 나만의 기능 추가
  - 설정시 인풋 빈값일 때 저장하기 버튼 비활성화 추가
  - 타이머 완료시 shake 애니메이션 추가
